- При першому завантаженні: зірка обертається 360° (0.8s)

## 2️⃣ ПЛАВАЮЧА НАВІГАЦІЯ (Sticky)

### Макет
```
┌─────────────────────────────────┐
│                                 │
│  [💳]  Купити    [✨] Активувати│
│   сертифікат      сертифікат    │
│  ─────────       ─────────────  │
│                                 │
└─────────────────────────────────┘
```

### Специфікація
- **Позиція:** Sticky, 140px від верху (під header)
- **Фон:** Білий з тінню
- **Таби:**
  - Ширина: 50% кожен
  - Висота: 80px
  - Іконка: 32px (зверху)
  - Текст: 16px, SemiBold
  - Відступи: 12px всередині

### Стани
- **Активний:**
  - Фон: світло-фіолетовий (#DDD6FE)
  - Текст: темно-фіолетовий (#6D28D9)
  - Нижня лінія: 3px фіолетовий (#8B5CF6)
  
- **Неактивний:**
  - Фон: білий
  - Текст: сірий (#6B7280)
  - Hover: світло-сірий фон

### Анімація при переключенні
1. Нижня лінія переміщується до нового табу (0.3s ease)
2. Іконка активного табу: легкий підскок вгору (0.3s)
3. З'являються зірочки навколо іконки (5-7 штук, розлітаються, 0.5s)
4. Контент форми: fade-out старий → fade-in новий (0.4s)

### Запам'ятовування
- Останній вибір зберігається в localStorage
- При поверненні на сайт відкривається останній обраний таб

## 🎬 АНІМАЦІЇ (Детально)

### Глобальні принципи
1. **БЕЗ scroll-trigger** - тільки при взаємодії
2. **Timing:** ease-out для появи, ease-in для зникнення
3. **Тривалість:** 0.3-0.8s (швидко, щоб не нудьгувати)
4. **Зірочки:** завжди розлітаються від точки дії

### Зірочки (Магічний ефект)

#### Параметри
```css
.star {
  size: 8px - 16px (випадково)
  color: #F59E0B (золотий)
  shape: ⭐ (SVG)
  count: 5-7 за раз
}
```

#### Поведінка
1. З'являються в точці дії (scale 0 → 1)
2. Розлітаються в різні сторони (радіально)
3. Обертаються при русі (360°)
4. Поступово зникають (opacity 1 → 0)
5. Тривалість: 0.6-0.8s

#### Використання
- При виборі номіналу
- При натисканні кнопок
- При переключенні табів
- При виборі партнера
- При успішній активації

### Анімація кнопок

#### Hover
```css
transform: scale(1.02)
box-shadow: 0 8px 16px rgba(139, 92, 246, 0.2)
transition: 0.2s ease
```

#### Active (натискання)
```css
transform: scale(0.98)
/* + зірочки розлітаються */
```

#### Success (після успіху)
```css
background: зелений
галочка ✓ з'являється
зірочки по всій кнопці
```

### Анімація форм

#### Поява помилки
```css
border: червоний
shake анімація (тремтіння)
текст помилки slide-down
іконка ❌ з'являється
```

#### Успішне заповнення
```css
border: зелений
галочка ✓ з'являється
легке свічення
```

### Перехід між кроками

#### Купівля: Форма → Редирект
1. Форма fade-out (0.3s)
2. Лоадер з'являється (0.2s)
3. Зірочки обертаються навколо лоадера
4. Текст: "Перенаправлення на оплату..."
5. Редирект на Monobank

#### Активація: Крок 1 → Крок 2
1. Результати перевірки fade-in (0.4s)
2. Кнопка "Продовжити" пульсує (привертає увагу)
3. При натисканні:
   - Форма Кроку 1 slide-left + fade-out (0.4s)
   - Форма Кроку 2 slide-right + fade-in (0.4s)
   - Прогрес-бар оновлюється (50% → 100%)

### Магічне перетворення (Головна анімація)

#### Тригер
Показується в інфографіці при тапі на крок 2-3

#### Сценарій
```
Frame 1 (0s):
  Сертифікат статичний

Frame 2 (0.3s):
  Сертифікат пульсує (scale 1 → 1.1 → 1)

Frame 3 (0.5s):
  Зірочки з'являються навколо
  Початок обертання зірок

Frame 4 (0.8s):
  Зірочки утворюють вихор
  Сертифікат починає розмиватись (blur + opacity)

Frame 5 (1.2s):
  Сертифікат зник
  З центру вихору з'являється подарунок (scale 0 → 1)

Frame 6 (1.5s):
  Зірочки розлітаються
  Подарунок набуває кольору

Frame 7 (1.8s):
  Золоте свічення навколо подарунка
  Зірочки повністю зникли

Frame 8 (2.2s):
  Подарунок статичний
  Легке мерехтіння триває
```

#### Технічна реалізація
```css
@keyframes magicTransform {
  0% { /* Сертифікат */ }
  30% { /* Пульсація */ }
  50% { /* Зірочки з'являються */ }
  70% { /* Вихор + розмиття */ }
  85% { /* Подарунок з'являється */ }
  100% { /* Фінальне свічення */ }
}
```
